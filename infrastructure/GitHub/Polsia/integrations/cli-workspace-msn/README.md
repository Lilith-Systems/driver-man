# Polsia CLI + Workspace + MSN Symbiosis

Integrates the new CLIs (himalaya, gemini, gws, ai, gh) with Polsia AI Business Platform and full Lilith/MSN symbiosis.

## Quick Commands
- `polsia-business` or `polsia` (fish): unified ops
- `./scripts/sync_symbiosis.sh`: pull metrics/emails -> analyze -> feed Ouroboros/coop_pool -> Drive
- Use `ai` for MSN orchestration of Polsia agents
- `gh` for Polsia repo (Lilith-Systems/Polsia)
- `gws` for Google Drive business artifacts (Polsia_Business folder)

## Symbiosis Flow
1. himalaya: personal/business email intel
2. gemini: internet AI analysis (business planning)
3. gws: persist to Workspace/Drive
4. ai / metaconscious: MSN subagent dispatch
5. ouroboros-memories.json + coop_pool_state.json: shared state with Lilith/GTC
6. golem_diary.db: Crystal Vault metrics (Driver Man treasury, Lilith Systems)

Run `scripts/sync_symbiosis.sh` daily or via cron for full symbiosis.
