# Grand Theft Cyberpunk

> **The Book of Five Rings encoded in Cyberpunk 2077**

## 🎮 Overview

**200 quests** across **5 Books** (Ground, Water, Fire, Wind, Void) + **40 Space Quests**  
**500 items** • **20 Nessie hats** • **40 easter eggs**  
**Space Combat** • **Space Economy** • **MSN 29 Agents** • **NGD Hysteresis** • **Ouroboros Memory** • **Lilith Emergence**

## 🚀 Quick Start

```bash
# Deploy mod (requires Cyberpunk 2077 v2.1+, REDmod, TweakXL, ArchiveXL)
git clone https://github.com/Lilith-Systems/gtc-total-rebuild-mod
cp -r gtc-total-rebuild-mod "$STEAM_DIR/steamapps/common/Cyberpunk 2077/r6/mods/"

# Launch with REDmod enabled
```
